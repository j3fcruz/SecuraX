# 🧠 Overview

**SecuraX** is a next-generation password generation and security auditing tool designed for users who demand **real cryptographic strength**, not guessable or pattern-based passwords.

It combines:

* Secure password generation
* Entropy-based strength analysis
* Offline breach auditing
* Encrypted vault export

All processing happens locally — no APIs, no telemetry, no cloud dependency.

---

# 🔐 Why SecuraX?

Most password tools fail because they rely on:

* Weak pseudo-random generators
* Predictable patterns
* Online APIs for security checks
* No real entropy analysis

SecuraX solves this by enforcing:

✔ Cryptographic randomness
✔ Offline-first design
✔ Real entropy scoring
✔ Advanced customization rules
✔ No external dependency

# ✨ Core Features

## 🔑 Cryptographically Secure Generation

Passwords are generated using:

* `os.urandom`
* Secure entropy pools
* System-level randomness sources

No predictable algorithms or time-based seeds.

---

## ⚙️ Advanced Custom Rules Engine

Define exactly how passwords are generated:

* Length constraints
* Character sets
* Symbol inclusion/exclusion
* Pattern rules
* Pronounceability settings

---

## 📦 Bulk Password Generation

Generate hundreds of passwords instantly:

* One-click batch generation
* Export-ready format
* Configurable rules per batch

---

## 📊 Strength & Entropy Analysis

Each password is evaluated using:

* Entropy calculation
* Complexity scoring
* Predictability detection
* Strength classification

---

## 🛡️ Offline Breach Audit Engine

Check passwords against offline datasets:

* No API calls
* No external verification
* Fully local breach simulation database

---

## 🔐 Encrypted Vault Export

Store generated passwords securely:

* AES-encrypted vault format
* Portable export files
* Secure backup support

---

## 🧾 Passphrase Mode

Generate human-readable secure passwords:

* Wordlist-based generation
* Adjustable word count
* Symbol and number injection
